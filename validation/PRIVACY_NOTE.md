# Datenschutzhinweis zur Release-Validierung

Die Validierungsdateien enthalten nur technische Zustände, generische Entity-IDs, Prüfergebnisse und anonymisierte Messwerte. Sie enthalten keine Zugangsdaten, Tokens, privaten Adressen, Standortkoordinaten oder vollständigen Home-Assistant-Konfigurationen.

Runtime-Werte stammen aus einer lokalen Testinstallation und dienen ausschließlich als Nachweis der geprüften Logik. Installationsspezifische Entity-Zuordnungen bleiben Sache der jeweiligen Home-Assistant-Instanz.

Die vollständigen Rohreports der Referenzinstanz werden nicht veröffentlicht,
weil sie konkrete Zeit-, Wetter- und Verbrauchswerte enthalten. Im Repository
liegt ausschließlich eine zusammengefasste Abnahme ohne Host, Token oder lokale
Entity-Mappings.
